# Urban Heat Island Effect vs. Jaipur Energy Grid

## Project Overview
This project investigates the correlation between the Urban Heat Island (UHI) effect and power grid strain in Jaipur, Rajasthan. By analyzing temperature data and energy consumption patterns, this project identifies critical temperature thresholds that impact infrastructure.

## Tech Stack
* *Language:* Python
* *Libraries:* Pandas, NumPy, Matplotlib, Seaborn
* *Data Cleaning:* Handling missing data and statistical normalization
* *Version Control:* Git & GitHub

## Key Insights
* The analysis reveals a significant spike in grid load once the maximum daily temperature exceeds *42°C*.
* The project successfully processed 3 years of synthetic historical data to simulate real-world infrastructure stress.
